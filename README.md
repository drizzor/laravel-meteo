## A propos du projet

Application créée avec Laravel, Tailwind et Livewire. Les données météorologiques sont récupérées avec OpenWeatherMap avec le client HTTP intégré. L'utilisateur peut choisir la ville souhaitée et les données sont chargées dynamiquement.
